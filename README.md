# Kuverse

At its core, a Ku is more than just a digital collectibleâ€”it's a companion, an adventure buddy, and a cherished baby kanary in the world of Kuverse. ## KU ORE

KU ORE (KORE) is a mysterious KU particle that can be obtained by hatching a KU EGG or by defeating a KU.

## How to Run this project

### Install backend dependencies

```bash
npm install
```

### Install client dependencies

```bash
cd client
npm install
```

### Run both Express & React from root project-directory

```bash
npm run dev
```
